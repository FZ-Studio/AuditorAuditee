# AuditorAuditee
BungeeCord审核员与受审核人
### 指令
##### 添加审核员：
aa add auditee 受审核人 auditor 审核员1 审核员2 审核员3...
##### 移除审核员：
aa remove auditee 受审核人 auditor 审核员1 审核员2 审核员3...
##### 移除受审核人的所有审核员：
aa remove auditee 受审核人
##### 人员名单
aa list
